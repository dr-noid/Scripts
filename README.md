# Scripts

My keyboard bindings and scripts used on my GMMK Pro

## QMK Changes

### Keybinds

* fn + del = f13
* fn + pgup = f14
* fn + pgdn = f15
* fn + end = f16

### RGB

* fn + up = cycle rgb up
* fn + down = cycle rgb down
* fn + right = shift rgb saturation?
* fn + left = shift rgb saturation?

### Extra

* fn + escape = enter bootloader state (not sure about this one)
